
1. Utilisation de Ant

Création du repertoire bin
$ ant init

Supprimer le répertoire bin
$ ant clean

Compilation du projet
$ ant compile

Démarer le serveur
$ ant startServer

Créer le JAR distribuable du client
$ ant distclient
